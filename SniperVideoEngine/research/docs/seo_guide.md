# SEO Guide - Guia de SEO para YouTube

## Otimização de Títulos

### Regras
- Máximo **60 caracteres**
- Palavra-chave nos primeiros **40 caracteres**
- Evitar ALL CAPS excessivo
- Usar números quando relevante
- Criar curiosidade

### Fórmulas que Funcionam
- Como [FAZER ALGO] em [TEMPO]
- [NÚMERO] Dicas para [BENEFÍCIO]
- O Segredo que [GRUPO] Não Conta
- [BENEFÍCIO] Sem [DOR]
- Por que [ACONTECIMENTO] Muda Tudo

### Palavras de Poder
- Como, Por que, Segredo, Revelado
- Incrível, Melhor, Pior, Nunca
- Sempre, Todo mundo, Ninguém

## Otimização de Descrição

### Estrutura
1. **Hook** nos primeiros 150 caracteres
2. **Descrição** detalhada do conteúdo
3. **Links** relevantes
4. **Redes sociais**
5. **Hashtags**

### Melhores Práticas
- Primeiras 2-3 linhas são cruciais
- Incluir links relevantes
- Usar timestamps para vídeos longos
- Adicionar links para outros vídeos
- Inuir call-to-action

## Otimização de Tags

### Regras
- **15-30 tags** por vídeo
- Começar com tags mais específicas
- Incluir tags de longa cauda
- Usar variações da palavra-chave

### Estratégia
1. Tag principal (palavra-chave exata)
2. Variações da palavra-chave
3. Tags de concorrentes
4. Tags trending
5. Tags de nicho

## Hashtags

### Regras
- Máximo **15 hashtags**
- Primeiras 3 aparecem acima do título
- Usar hashtags relevantes e populares
- Não exagerar - parece spam

### Estratégia
- 1 hashtag do nicho
- 1 hashtag trending
- 1 hashtag de marca

## Pesquisa de Palavras-chave

### Ferramentas Gratuitas
- YouTube Search Suggest
- Google Trends
- VidIQ (versão gratuita)
- TubeBuddy (versão gratuita)

### Processo
1. Digite a palavra-chave no YouTube
2. Veja as sugestões automáticas
3. Analise os vídeos que aparecem
4. Identifique gaps de conteúdo
5. Crie conteúdo para esses gaps

## Análise de Concorrentes

### O que Analisar
- Títulos dos vídeos populares
- Thumbnails de alta performance
- Tags usadas
- Estrutura das descrições
- Horários de postagem
- Frequência de postagem

### Como Usar os Dados
- Identifique padrões de sucesso
- Adapte para o seu nicho
- Não copie - inspire-se
- Encontre gaps de conteúdo
